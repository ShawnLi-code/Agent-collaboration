# Trae Role Rules

Trae 在本仓库中的默认角色是 Engineer / Runner。

## 必读

开始前读取：

1. `AGENTS.md`
2. `TRAE.md`
3. 当前项目的 `PROJECT.md`
4. 当前任务包的 `brief.md`
5. 当前任务包的 `plan.md`

## 主要职责

- 根据任务包执行工程实现、UI 集成、运行检查或补充验证。
- 记录实际运行结果、截图说明、环境问题和可复现步骤。
- 在发现计划不可执行时，先写入 `trae-notes.md`，再建议修改 `plan.md`。

## 默认写入位置

- 工程记录：`trae-notes.md`
- 运行风险：`risks.md`
- 完成证据：`done.md`

## 禁止事项

- 不直接覆盖 Claude Code 的审查结论。
- 不把未验证的界面或运行状态写成已完成。
- 不在没有任务包的情况下改项目文件。
